# 项目名及简介
* 基于appium+python2.7封装的稳定性自动化测试工具
* 作者：李然

# 介绍
* android稳定性自动化工具
* 优于monkey的自动化实现
* 同步输出运行log和设备log文件，本地化多机日志管理
* 基于页面id进行检索，并进行随机操作
* 支持多设备andoird并行
* 支持app崩溃重启机制、失败重连，优于monkey 
* 检测当前页是否为被测app，如不是自动重新进入
* 可自行设置执行操作次数



## 命令运行

```
python appium-monkey-liyyq.py
```
